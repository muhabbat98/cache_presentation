Kesh qanday ishlaydi? (Flow) 🔄

Foydalanuvchi so'rov yuborganda, brauzer avval keshni tekshiradi.

  • Hit: Ma'lumot keshdan topildi (Chaqmoqdek tez! ⚡️).
  • Miss: Kesh bo'sh, serverga borish kerak (Sekinroq 🐢).

HTTP Headers (Asosiy qurol) 🛡️


// Cache-Control turlari:
Cache-Control: max-age=31536000 // 1 yil saqla
Cache-Control: no-store         // Hech narsani saqlama
Cache-Control: no-cache         // Saqla, lekin har doim tekshir

ETag: Faylning "barmoq izi".

Amaliy Kod: Cache API 💻


self.addEventListener('fetch', (event) => {
  event.respondWith(
    caches.match(event.request).then((res) => {
      // Cache First Strategiyasi
      return res || fetch(event.request);
    })
  );
});
